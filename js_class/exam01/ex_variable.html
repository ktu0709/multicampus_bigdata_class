var 변수명    ---> 전역  es6 이전  : 함수레벨 스코프 
           let 변수명    ---> 지역  es6 이후  : 블럭 레벨 스코프
           const 변수명  ---->상수  es6 이후   : 블럭 레벨 스코프
           스코프 scope : 메모리에 변수가 올라가고 내려오는 기준 또는 범위 
    
      window(전역객체) : 자바스크립트에서 사용하는 변수, 함수, 객체 속성등 
                        프로그램에서 제공해주는 전역객체 
         -   tip:  전역객체는  브라우저가 종료될때 까지 메모리에서 소거되지 않는 객체  
       
         -  전역에서 var로 선언한 변수는 전역객체인 window 속성으로 등록이 된다.  
         -  let, const 로 선언한 변수는 전역객체인 window 속성으로 등록되지 않는다.
