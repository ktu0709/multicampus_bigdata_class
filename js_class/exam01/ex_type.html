primitive  type  : String , Number, Boolean, Null, Undefined   
          -> 데이터 복사할때 깊은 복사가 되는 타입

       reference  type : Object
          -> 데이터 복사할때 얕은 복사가 되는 타입 (주소 대입)

    
querySelector  : Css 스타일 기반요소를 선택하는 함수   -> 첫번째 요소 
                 ex) document.querySelector("#BB");   id = BB
                     document.querySelector(".note");  class  ='note'

getElementById  : id만 선택하는 함수 
            ex) document.getElementById("BB"); id = BB
